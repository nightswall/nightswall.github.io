Main Characteristics

DIONA provides Deep Learning integrated Network Security via monitoring applications.

  • Anomaly detection on Network Flow

    DIONA monitors network flow with rules and a DL engine to analyze the traffic status.

  • Anomaly detection on IoT Telemetry

    DIONA monitors IoT telemetries and learn from them. It uses previous knowledge to decide on an behavioral result.

  • Fully clustered

    The better part yet, DIONA is fully dockerized!

Tech Stack

Our tech stack includes popular technologies such as Kubernetes, Docker and Python, along with specialized tools like TensorFlow for deep learning and Grafana for data visualization. We also use PyPSA and Pandas data simulation and manipulation for the sake of training of our DL engines. Together, these tools help us build robust and scalable solutions for IoT network security and behavioral analysis.
